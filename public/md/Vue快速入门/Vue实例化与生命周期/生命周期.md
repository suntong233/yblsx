# 生命周期钩子函数

![lifestyle](./lifecycle.png)

* 生命周期钩子函数

```js
    new Vue({
        el: '#app',
        data: {
            msg: 'hhh'
        },
        let app = new Vue({
        el: '#app',
        data: {
            msg: 'aa'
        },
        beforeCreate() {
            console.log(this.msg)   // undefined
            console.log('beforeCreated')
        },
        created() {
            console.log(this.msg)    // aa
            console.log('created')
        },
        beforeMount(){
            console.log('beforeMount')
        },
        mounted(){
            console.log('mounted')
        },
        beforeUpdate(){
            console.log(this.msg)   // 改变数据的值
            console.log('beforeUpdate')
        },
        updated(){
            console.log(this.msg)   // 改变数据的值
            console.log('updated')
        },
        beforeDestroy(){
            console.log('beforeDestroy')
        },
        destroyed(){
            console.log('destroyed')
        }
    })
```
